# 6. NIEUWE: lib/utils/epc_qr_code.dart — port van CalyCompta epcQrCode.ts naar Dart

Pure utility:
```dart
String buildEpcQrPayload({
  required String iban,
  required String beneficiary,
  required double amount,
  required String structuredCommunication,  // OGM display +++/+++
  String? bic,
}) {
  // EPC069 v002 SCT format — 11+ lines newline-separated
  // BCD\n002\n1\nSCT\n[BIC]\n[Beneficiary]\n[IBAN]\nEUR[amount.toStringAsFixed(2)]\n[Purpose]\n[StructuredRef]\n[Remittance]\n
}
```

Lees CalyCompta's epcQrCode.ts en kopieer 1-op-1 — dezelfde format moet uitkomen.
